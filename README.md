# AWTRIX Python Integration

AWTRIX is a modular Python framework for integrating and displaying data from various sources on AWTRIX smart clocks. It uses a plugin-based architecture, allowing you to easily add new features such as weather updates, YouTube subscriber counts, and productivity tools like Pomodoro timers.

## Features

- **Plugin-based:** Easily extend functionality by adding new plugins.
- **Weather Plugin:** Displays current weather using OpenWeatherMap.
- **YouTube Plugin:** Shows live subscriber counts for a channel.
- **Pomodoro Plugin:** (Coming soon) Productivity timer for focused work sessions.
- **Easy Configuration:** Manage plugins and settings via a JSON config file.

## Requirements

- Python 3.12+
- AWTRIX smart clock with Custom API enabled
- [requests](https://pypi.org/project/requests/)
